# Food Delivery Management System

## Overview

This project is a SQL-based implementation of a Food Delivery Management System developed using MySQL. It focuses on solving common business scenarios through database design, data manipulation, reporting, and database programming.

The project reflects practical SQL usage across multiple areas including schema design, business reporting, data analysis, window functions, stored procedures, and triggers. It serves as part of my Data Analytics portfolio, demonstrating structured SQL problem-solving using a real-world business domain.

---

## Business Scenario

A food delivery platform needs a database to manage customers, restaurants, delivery partners, and customer orders while supporting day-to-day business operations and reporting requirements.

The database should be capable of:

- Managing customer, restaurant, and order information.
- Supporting business reporting and revenue analysis.
- Tracking customer spending and restaurant performance.
- Maintaining data integrity through constraints and triggers.
- Automating common database operations using stored procedures.
